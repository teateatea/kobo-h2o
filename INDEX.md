# Project Index

.claude/TASKS.md - cross-session backlog of all plugin tasks, numbered and sectioned by area
.claude/settings.local.json - Claude Code local permissions for this project
.claude/plans/ - plan files for all features (DISCUSSION, APPROVED, PREFECT-APPROVED, VERIFIED prefixes)
.gitignore - excludes node_modules, dist, .claude/, and test SQLite files from git
KoboReader.sqlite - test/sample Kobo database (gitignored, local only)
KoboReader260319.sqlite - additional test Kobo database (gitignored, local only)
LICENSE - MIT license
README.md - user-facing plugin documentation, install and usage instructions
deploy.mjs - copies built plugin files (main.js, manifest.json, sql-wasm.wasm) to an Obsidian vault
  also: reads vault path from KOBO_VAULT env var or a hardcoded default
esbuild.config.mjs - esbuild bundler config; bundles src/main.ts -> main.js, copies sql-wasm.wasm to plugin root
  also: dev mode runs in watch mode, production mode minifies and exits
main.js - compiled/bundled plugin output (do not edit; regenerated by npm run build)
manifest.json - Obsidian plugin manifest (id, name, version, minAppVersion, isDesktopOnly)
node_modules/ - npm dependencies, not project code
package.json - npm project config; defines dev/build/deploy scripts and dependencies (sql.js, esbuild, typescript)
package-lock.json - npm lockfile
setup.bat - Windows helper script for setting KOBO_VAULT environment variable
sql-wasm.wasm - WebAssembly binary for sql.js SQLite engine (copied from node_modules at build time)
src/main.ts - Obsidian plugin entry point: registers commands, orchestrates import flows, writes notes to vault
  also: BookSelectionModal (choose which books to import), OverwriteConfirmModal, device detection across Windows/macOS/Linux
src/renderer.ts - generates Markdown note content from KoboBook/KoboHighlight data using user-defined templates
  also: frontmatter, note heading, highlight, annotation, footer, append block, and My Words rendering; author/title/date/percent formatting helpers
src/settings-tab.ts - Obsidian settings UI: all plugin settings panels, collapsible Advanced Formatting groups, template variable pill hints, date format live preview
src/sqlite-parser.ts - reads KoboReader.sqlite via sql.js; queries highlights, books, shelves (collections), and My Words (WordList)
  also: lazy-loads sql-wasm.wasm; cleans raw chapter titles; exposes clearSqlJsCache
src/template-parser.ts - pre-processes conditional template syntax ({{tag|N}} / {{N|text}}) before variable substitution
  also: parse-cache for compiled AST nodes; returns plain {{var}} template for applyVars pipeline
src/text-parser.ts - parses plain .txt, Kobo clippings format, and .csv/.tsv highlight export files into KoboBook arrays
src/types.ts - TypeScript interfaces (KoboHighlight, KoboWord, KoboBook, KoboImporterSettings) and DEFAULT_SETTINGS
src/utils.ts - sanitizeFilename: strips illegal filename/YAML characters and normalises whitespace
tsconfig.json - TypeScript compiler config
